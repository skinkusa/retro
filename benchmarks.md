# Match Engine Benchmarks (1990s Context)

Target statistics for a typical 90-minute match simulation. These benchmarks are used to tune the match engine for realism.

| Stat                | Typical Range | Ideal Target  |
| ------------------- | ------------- | ------------- |
| **Shots**           | 8 – 16        | **11–13**     |
| **Shots on Target** | 3 – 6         | **4–5**       |
| **Goals**           | 0.9 – 1.6     | **1.2 – 1.4** |
| **Possession**      | 35 – 65%      | balanced      |
| **Corners**         | 3 – 7         | **5**         |
| **Fouls**           | 8 – 15        | **11–12**     |
| **Yellow Cards**    | 1 – 3         | **1.8**       |
| **Red Cards**       | 0 – 0.2       | **0.05–0.1**  |
| **Offsides**        | 1 – 3         | **2**         |
| **Saves (GK)**      | 2 – 5         | **3–4**       |

*Note: Goals are per team, so total match goals should average ~2.6.*
