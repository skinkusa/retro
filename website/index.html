Digital Nostalgia

Retro Manager 1993

A playable browser-based football management sim built with modern Next.js, but styled like a DOS-era tactical terminal.

Play the Game Host on Vercel

Why this feels like '93

  • Quad-division league structure, board expectations, and week-by-week news flashes.
  • Retro HUD components, grainy glass panels, and chunky typography inspired by early OS shells.
  • Text-based match engine, decision-heavy transfers, and persistent localStorage saves.

Features

Live Career Panel

Navigate club finances, staffing, and the tactical board while the toaster keeps you informed every week.

Transfer Negotiations

List players, respond to bids, and sign new talent with a simple retro menu system.

Match Simulation

Simulate each week, watch the text commentary roll by, and fast-forward entire seasons when you need a break.

Local Storage Saves

Saves live in your browser under retro_manager_save_v1.9.3 so you can continue careers on any Vercel deployment.

Deploy on Vercel

Push to GitHub, connect the repo to Vercel, and every commit will build a preview. A production deployment runs npm run build followed by npm run start. There are no serverless functions required.

Share the playable URL on socials and encourage players to bookmark the tab—saving their career is just a click away.
