# Retro Manager landing site

This folder contains a single-file marketing landing page (`index.html`) plus styles (`styles.css`). It can be hosted as a static site (for example, another Vercel project or the same project’s `/public` folder) to promote the playable experience. The landing page highlights key features, links to the live game, and explains why localStorage saves are important.

When hosting on Vercel, point a secondary project at this folder (set the output directory to `website/`), then point the CTA to the root Vercel deployment (`https://retro-manager-1993.vercel.app` or your custom domain).
