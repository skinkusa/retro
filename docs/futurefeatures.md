RETRO MANAGER – FEATURE EXPANSION PLAN

This document lists additional features to build on top of the existing
Retro Manager systems. Current systems already implemented include:
match engine, tactics, dynamic attributes, player development, staff
system, transfers, finances, stadium upgrades, league structure, news
feed, statistical hub, and save system.

The features below expand immersion, world simulation, and player
identity.

================================================== MATCHDAY ENHANCEMENTS
==================================================

Momentum Meter Displays which team currently has momentum during a
match.

Example: Momentum Home ███████░░░░ Away ███░░░░░░░

Momentum increases chance creation and reflects sustained pressure.

Injury Time Drama Increase unpredictability after the 90th minute. -
Defensive errors slightly increase - Long balls increase - Chance of
dramatic late goals rises

Example commentary: 90+2’ GOAL! Heartbreak for the visitors!

Derby Atmosphere Modifier Rivalry matches receive modifiers: - Higher
aggression - Higher attendance - Stronger home advantage

Example: Derby Day Atmosphere: Electric

================================================== PLAYER IDENTITY
IMPROVEMENTS ==================================================

Player Portrait Generator Procedurally generated retro-style faces.
Parameters: - Hair style - Facial hair - Skin tone - Age - Nationality
hints

Benefits: Creates memorable players and stronger identity.

Player Radar Ability Chart Visual stat display for player abilities.

Example: Pace ▲ Passing ◀●▶ Shooting ▼ Stamina

Used in: - Player profiles - Transfers - Scouting



Examples: Carlos “El Toro” Diaz John “The Hammer” Smith

Triggered by: - High scoring - Long careers - Fan popularity

================================================== PLAYER BEHAVIOUR
SYSTEMS ==================================================

Hot / Cold Streaks Players can enter form streaks.

Example: Hot Streak Last 5 matches: 4 goals +5% attacking performance

Cold streaks reduce performance slightly.

Player Personality Traits Hidden attributes: - Professional - Lazy -
Leader - Temperamental - Big Match Player - Injury Prone

Traits influence: - Development - Morale - Consistency

================================================== CLUB IDENTITY SYSTEMS
==================================================

Rivalries Predefined rival clubs. Effects: - Derby matches - Media
hype - Increased attendance

Example news: Derby Day approaching! City vs United this weekend.

Fan Mood System Fans react to team performance.

Levels: Furious Concerned Happy Delighted

Impacts: - Attendance - Board pressure

Attendance System Attendance determined by: - Club form - Rivalries -
Fan mood - Stadium capacity

Example: Attendance: 18,440 Capacity: 22,000

================================================== FINANCIAL ADDITIONS
==================================================

Sponsorship Deals Clubs receive yearly sponsorships.

Example: Shirt Sponsor $1.4M per season

Better teams attract larger sponsors.

================================================== WORLD SIMULATION
==================================================

Manager Sackings (AI Teams) Other clubs can fire managers.

Example news: Chelsea sack manager after poor run.

New Manager Bounce New managers provide short-term morale boosts.

Example: New manager bounce Team morale +5

Regional Talent Pools Different regions generate players with natural
strengths.

Example: Brazil → +skill Germany → +stamina England → +heading
